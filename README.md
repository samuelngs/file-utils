# file-utils
Files packer, Decrypt and Encrypt files

##### Pack files within a folder into a single (extract) shell script
```
$ ./packer {folder_name}
```

##### Decrypt file
```
$ ./decrypt {filename} {private_key}
```

##### Encrypt file
```
$ ./encrypt {filename} {public_key}
```

##### Generate private and public key
```
$ ./generate-key
```
